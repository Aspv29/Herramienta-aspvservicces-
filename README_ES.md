## 🚀 Descripción

**AspvServicces** es una herramienta profesional premium para técnicos de servicio móvil que ofrece una suite completa de utilidades para desbloqueo, reparación y servicio de dispositivos Android e iOS.

## ✨ Características Principales

### 📱 Herramientas Android

- **FRP Bypass (Factory Reset Protection)**
  - Múltiples métodos: ADB, Fastboot, Combination, Test Point
  - Soporte para todas las marcas principales
  - Samsung, Xiaomi, Huawei, Motorola, Oppo, Vivo, Realme, Tecno, Infinix

- **Eliminación de Bloqueos de Pantalla**
  - PIN, Patrón, Contraseña, Huella Digital
  - Face Recognition

- **Flasheo de Firmware**
  - Odin (Samsung)
  - SP Flash Tool (MediaTek)
  - QFIL (Qualcomm)
  - HiSuite (Huawei)
  - Mi Flash (Xiaomi)

- **MDM / Knox Guard Removal**
  - Samsung Knox Guard
  - Google EMM/MDM
  - VMware Workspace ONE
  - Microsoft Intune
  - MobileIron

- **Reparación de IMEI**
  - Soporte para chipsets MediaTek y Qualcomm
  - Backup de EFS/QCN

- **Bypass de Mi Account (Xiaomi)**
  - Múltiples métodos de bypass
  - Compatible con últimas versiones de MIUI

- **Unlock/Lock Bootloader**
  - Desbloqueo de bootloader
  - Soporte para múltiples marcas

### 🍎 Herramientas iOS

- **iCloud Bypass**
  - Métodos: checkra1n, F3arRa1n, Sliver, Signal, Mina
  - Soporte para múltiples versiones de iOS

- **Activación de Dispositivos**
  - Activación directa
  - Bypass de activación

- **Jailbreak**
  - Recomendaciones según versión de iOS
  - Soporte para herramientas populares

### 📡 Desbloqueo de Operadores (México)

- **Telcel**
  - Generación de códigos de desbloqueo
  - Eliminación de APKs de bloqueo
  - Método NVRAM (avanzado)

- **AT&T**
  - Método oficial
  - Eliminación de apps de operador
  - Bypass de restricciones

- **Payjoy**
  - Eliminación de bloqueo Payjoy
  - Múltiples métodos de bypass
  - **Advertencia**: Usar solo en dispositivos pagados

### 🛡️ MDM Bypass con QR

- Generación de códigos QR para bypass de MDM
- Soporte para múltiples carriers mexicanos
- Compatible con Samsung, LG, Motorola, y dispositivos genéricos

### 💿 Gestión de Drivers

- **Drivers incluidos:**
  - Android ADB & Fastboot
  - Samsung USB Drivers
  - Qualcomm QDLoader HS-USB
  - MediaTek VCOM
  - Apple Mobile Device Support
  - Huawei HiSuite
  - LG USB Drivers
  - Motorola Device Manager

### ⌨️ Terminal Premium

- Interfaz de terminal con colores premium
- Ejecución de comandos ADB/Fastboot
- Comandos rápidos predefinidos
- Output con formato mejorado

## 📋 Requisitos del Sistema

- **Sistema Operativo**: Windows 10/11 (64-bit)
- **RAM**: Mínimo 4GB
- **Espacio en Disco**: 500MB
- **Permisos**: Administrador (para instalación de drivers)
- **.NET Framework**: 4.7.2 o superior

## 🔧 Instalación

1. Descarga el instalador `AspvServicces-Setup.exe`
2. Ejecuta como Administrador
3. Sigue el asistente de instalación
4. Instala los drivers necesarios desde la sección de Drivers
5. ¡Listo para usar!

## 📖 Uso Básico

### Modo GUI

1. Ejecuta AspvServicces desde el acceso directo
2. Conecta tu dispositivo por USB
3. Haz clic en "Detectar Dispositivos"
4. Selecciona la herramienta deseada del menú lateral
5. Sigue las instrucciones en pantalla

### Modo CLI

```bash
# Abrir terminal de AspvServicces
aspv-cli

# O desde la instalación
cd "C:\Program Files\AspvServicces"
node src/cli.js
```
